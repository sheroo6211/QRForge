# QRForge

Free QR code generator. No login or installation needed. Works in any browser.

## Features
- QR codes for URLs, text, email and phone
- WiFi QR code (scan to connect)
- Contact card QR code (vCard)
- Custom colors
- Download as PNG
- QR code history
